## Описание проекта

Репозиторий содержит материалы практического курса по применению методов Data Science и машинного обучения в медицинской сфере. Проект разработан в рамках образовательной программы **SkillFactory** и ориентирован на работу с биомедицинскими сигналами: ЭКГ (электрокардиограмма) и ЭЭГ (электроэнцефалограмма).

### Ключевые направления:
- **Классификация медицинских сигналов** с помощью машинного обучения
- **Предобработка и визуализация** временных рядов ЭКГ/ЭЭГ
- **Методы снижения размерности**: PCA, t-SNE
- **Автоматизированное машинное обучение (AutoML)**
- **Глубокое обучение**: CNN, LSTM для анализа сигналов
- **Вейвлет-анализ** для обработки нестационарных сигналов
- **Детекция эпилептических паттернов** в ЭЭГ

# Описание модулей курса

🔹 Часть 1: Введение и переобучение нейросетей

Загрузка и визуализация датасета Iris
Построение простых моделей классификации
Анализ переобучения: train/test split, кривые обучения
Базовые метрики качества моделей

🔹 Часть 2: Классификация ЭКГ с PCA и t-SNE

Предобработка сигналов ЭКГ
Извлечение признаков из временных рядов
Визуализация многомерных данных методами снижения размерности
Сравнение классификаторов: Logistic Regression, SVM, Random Forest

🔹 Часть 3: AutoML для медицинских данных

Автоматический подбор моделей и гиперпараметров
Использование библиотек: auto-sklearn, TPOT, H2O.ai
Интерпретация результатов AutoML-пайплайнов

🔹 Часть 4: Вейвлет-анализ сигналов ЭЭГ

Теория вейвлет-преобразований
Применение PyWavelets для декомпозиции сигналов
Выделение частотных компонентов (альфа, бета, гамма-ритмы)
Визуализация скалограмм

🔹 Часть 5: CNN для детекции эпилепсии

Архитектура свёрточных нейросетей для 1D-сигналов
Подготовка данных: сегментация, аугментация, балансировка классов
Обучение и валидация модели на датасете ЭЭГ
Анализ метрик: precision, recall, F1-score, ROC-AUC

🔹 Часть 6: Гибридные модели CNN+LSTM для классификации моторных образов

Комбинация CNN (извлечение признаков) и LSTM (временная зависимость)
Задача классификации моторных образов (Motor Imagery)
Построение end-to-end пайплайна глубокого обучения
Сравнение с традиционными методами (CSP + SVM)
